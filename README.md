# 🍡 Mochi Player

A cute, minimal local music player built with React + TypeScript + Vite.

Drop your music files and enjoy a clean, pastel-themed listening experience.

## Features

- 🎵 Drag & drop local music files (MP3, MP4, M4A, WAV)
- 🎨 Cute pastel pink/purple theme
- 📋 Playlist with track info and album art
- 🔀 Shuffle & repeat modes
- 🔊 Volume control
- 💾 Remembers your playlist and settings
- ⌨️ Space bar to play/pause
- 🎶 Mini equalizer visualizer

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React 19
- TypeScript
- Vite
- Zustand (state management)
- jsmediatags (ID3 tag reading)
- Plain CSS Modules
